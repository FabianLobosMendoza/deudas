# Tablero Financiero (Django)

Aplicación web en Django para organizar **deudas, ingresos, gastos y vencimientos** personales.  
Incluye un dashboard con resumen del mes, próximos vencimientos y formularios simples para cargar información.

---

## Requisitos previos

- Python 3.12+
- pip
- git
